#  DigiTools - Premium Digital Assets

A modern React-based web application for browsing and managing digital productivity tools. 

##  Features
- **Dynamic Product Grid**: Displays tools from a JSON data source.
- **Toggle View**: Switch between the product list and the shopping cart.
- **Cart Management**: Add products, remove items, and track total cost.
- **Toast Notifications**: Real-time feedback using react-toastify.
- **Responsive Design**: Built with Tailwind CSS and DaisyUI.
